## Overview

Manually exporting data is time-consuming, especially when reports can be automated on a schedule. To solve this, I built an automated **PDF Generation** workflow powered by **Gotenberg** (a free PDF tool) and a self-hosted **n8n**. 

This entire solution is fully automated, open-source, and free to operate.

## Technology

* **Language:** JavaScript
* **Automation Engine:** n8n (Self-hosted)
* **PDF Rendering:** Gotenberg
* **Integrations:** Google Cloud Services APIs
